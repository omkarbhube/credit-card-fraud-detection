# Credit Card Fraud Detection

Machine Learning project focused on detecting fraudulent credit card transactions on a highly imbalanced dataset.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Models Used
- Logistic Regression
- Random Forest Classifier

## Concepts Covered
- Imbalanced Classification
- Precision / Recall / F1 Score
- Threshold Tuning
- ROC-AUC
- Feature Scaling
- Data Leakage Prevention
- Class Weight Balancing
- Model Comparison

## Key Learnings
- Accuracy is misleading for imbalanced datasets
- Threshold tuning changes model behavior significantly
- Different models behave differently on fraud detection tasks
- Precision and recall tradeoffs are critical in real-world ML systems

## Best Model Performance

Random Forest -- Precision 0.941,  Recall 0.816,  F1 Score - 0.874

Kaggle Credit Card Fraud Detection Dataset:
